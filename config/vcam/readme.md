# Notes
Handheld (phone/tablet) and egocentric (Apple Vision Pro) capture data.

Only needed when generating vcam-type camera motions (`vcam=True`) from `be_generate_camera_animations.py`.

+ Setup
  + uncompress `captures.zip` to this folder

+ Folders
  + camera movement type > orientation > focal length [mm]
  + `14-28` focal length folders are copies of `28` folder
